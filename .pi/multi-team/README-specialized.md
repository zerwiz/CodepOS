# 🤖 Multi-Agent Specialized Teams

## 🎯 Specialized Agent Breakpoints

Each agent specializes in specific task domains and provides targeted help when needed.

---

## 📋 Available Agents

| Agent | Domain | Specialties |
|-------|--------|--------------|
| `design` | UI/UX | Wireframes, prototyping, accessibility |
| `ui-components` | Components | Component libraries, styling, animations |
| `api-backend` | Backend | API design, auth, routing |
| `testing` | QA | Unit tests, E2E, coverage |
| `integration` | Integration | Webhooks, events, 3rd party APIs |
| `database` | Data | Schema, migrations, optimization |
| `frontend` | UI | React pages, state, routes |
| `planning` | Architecture | System design, architecture |
| `setup` | Environment | Setup commands, scaffolding |

---

## 🚀 Usage Examples

### **Full System Orchestration**
```bash
just orchestrate
```

### **Get Help from Specialized Agents**
```bash
just design --help        # Design & wireframing help
just ui-components --help # Component creation help
just api-backend --help   # API design help
just testing --help       # Testing help
just integration --help   # Integration help
just database --help      # Database help
just frontend --help      # Frontend help
```

### **Targeted Requests**
```bash
just design new-page      # Design a new page
just ui-components button # Create button component
just api-backend users    # Design users API
just testing login        # Login tests
just integration webhook  # Setup webhook handler
just database schema      # Design schema
just frontend dashboard   # Dashboard page
```
