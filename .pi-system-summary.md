## 🔧 Key Features

### **📦 Extension Loading**
```javascript
export async function* load() {
  // Loads multi-agent system with council
  const multiAgents = await skills.spawnSubAgents();
  const councilMembers = await council.members;
  
  yield {
    name: "multi-agent-system",
    skills: skills,
    council: council,
    subAgents: multiAgents,
    councilMembers: councilMembers
  };
}
```

### **🤖 Multi-Agent Spawning**
```javascript
// Spawn sub-agents dynamically
async function spawnSubAgents() {
  for (const team of teams) {
    const agent = await spawnAgent(team);
    multiAgents.push(agent);
  }
  return multiAgents;
}
```

### **📑 Council Communication**
```javascript
await council.broadcast("system", "Orchestrator boot complete");
await council.request("security-scan", "run-scan");
```
